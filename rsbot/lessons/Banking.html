Banking

All of the methods associated with banking are contained in the Bank class (ctx.bank). The class contains methods for opening, closing, withdrawing, and depositing from the bank, as well as a Query for determining the contents.

Banking Methods

  • open(): opens the nearest bank. Returns true if it successfully opened the bank, false otherwise.
  • opened(): returns if the bank is currently open.
  • close(): closes the bank. Returns if it successfully closed the bank.
  • withdraw(int id, int amount): withdraws the specified amount of the item with the specified ID. Returns true if it withdrew the items, false otherwise.
    Note: passing 0 for amount with select the "Withdraw-All" option. Passing -1 will select the "Withdraw-All but one" option.
  • depositInventory(): clicks the "Deposit Inventory" button. Returns true if it clicked it.
