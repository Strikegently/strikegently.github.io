Interacting

Interacting with an entity, whether it is an Npc, a GameObject, or a Widget, will be at the core of every script you make. Everything that you can interact with extends Interactive, which supplies the interact() methods.

final Npc guard = ctx.npcs.select().name("Guard").nearest().poll();
guard.interact("Attack");
guard.interact("Attack", guard.name());

The Interaction Methods

  • interact(String action): interacts with the entity, right-clicking if necessary, and selecting the first menu option with the specified action. The action is what you are doing to the entity (e.g. "Attack" on a guard).
    Note: this always selects the first option available, so if there are multiple options with the same action (i.e. looting items on the ground) it will select the first menu option, not necessarily that of the specific entity you want. See interact(String action, String option)
  • interact(String action, String option): interacts with the entity, right-clicking if necessary, and selecting the first menu option with both the specified action and the specified option. The option is what you are performing the action upon (i.e. the name of the entity;
    e.g. interact("Take", "Bones")).
