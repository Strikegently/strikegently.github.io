ClientContext

ClientContext is the class that contains the API classes for interfacing with the game. Any time you need access information about the game, it will have to go through the ClientContext. ClientContext can be inherited from AbstractScript(the parent class of PollingScript) or from any class that extends ClientAccessor as the variable ctx.

Context Classes

  • backpack: contains methods for accessing information about the user's backpack, getting the items they have, how many items, etc.
  • bank: contains methods for accessing information about the user's bank (what items it contains, if it's currently open, depositing, withdrawing, etc)
  • camera: contains methods for turning the camera and changing the pitch.
  • chat: utility class for chat widgets (selecting chat options/continue)
  • combatBar: contains information about the combat bar, including activating abilities.
  • depositBox: has similar methods to bank, but for deposit boxes.
  • equipment: contains information about what the user has equipped.
  • game: contains information specifically related to the game state (if the user is logged in, the current status)
  • groundItems: GroundItems are items on the ground (duh). This contains methods for getting information about them.
  • hud: Hud is the RS3 ribbon (backpack, skills, etc), and contains methods to open/close each 'tab'
  • lobby: contains methods for traversing the lobby (logging in, changing worlds, etc)
  • movement: contains methods for moving/walking.
  • npcs: contains the methods for accessing in-game Npcs (Men, Guards, etc).
  • objects: contains the methods for accessing GameObjects (ladders, doors, etc).
  • players: contains methods for accessing information about players, including the user's player.
  • powers: contains information related to prayers and curses.
  • skills: contains information about skill levels and current experience
  • summoning: contains the methods for accessing information about summoning (if you have a familiar summoned, what familiar you have, your familiar's timer, etc).
  • widgets: contains information about widgets (in-game buttons)
