IntelliJ

IntelliJ is a full-featured, and very powerful, Java IDE. It is not required for RSBot development, but it is an option that is available. The Community edition is free (gratis), open-source, and more than enough for most developers, especially for those just seeking a tool for RSBot. After you download and install it, you can continue through this tutorial for setting up IntelliJ for use with RSBot.

Setting the Appropriate Language Level

The RSBot SDN compiles on Java 6, so it is recommended you do the same, to eliminate possible conflicts.
To set your language level:
  1. Go to File > Project Settings and select Project.
  2. Under Project SDK, select JDK 1.6. If you don't have it, you can download it from Oracle
  3. Just below, there is Project Language Level. Set this to 6.

Adding RSBot to Your Project

  1. While still on the Project Settings page, select Libraries from the left.
  2. Click the green + on the left-hand side and select the "Java" option.
  3. A file tree will open. Select the RSBot.jar and hit OK.
  4. Select OK / Apply

Setting Your Run Configurations

  1. At the top-right of the main window, there is a small button with a arrow. Click it, and select the "Edit Configurations" option.
  2. Click the green + at the top-left and select "Application"
  3. Set your main class to org.powerbot.Boot
  4. Set your VM Options to
    -Xmx384m -Xss2m -XX:CompileThreshold=1500 -Xincgc -XX:+UseConcMarkSweepGC-XX:+UseParNewGC
  5. Set your Program Arguments to -debug. This allows you to load local scripts.
  6. Check the "Use Alternate JRE" option. This must be a JRE, not JDK. OpenGL/DirectX will not work if it is a JDK.
  7. Select OK / Apply
